Projects

WAD File System

This project was part of my Operating Systems class. In this project, I developed my own file system for reading WAD files. This involved collecting and reading the data from user inputed WAD files, creating directories, and files; as well as asigning the needed data to those files. The user can then mount the system into a directory, then navigate and read data like they would normally.

For this project, I used C++, and the FuseApp library which allowed me to construct the file system.

Minesweeper

For this project, I made a functioning Minesweeper game from scratch in Visual Studio using C++. I did this without the help of any kind of game engine. This project was influential in teaching me C++.

The game creates a randomized level for you, with bombs in random locations. The adjacent tiles count how many bombs are near them and display their number when they are next to a revealed tile.
